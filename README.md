<br /> 



# Installation 

Option #1: <br>
```Press ctrl + shift + p``` **or** ```cmd + shift + p```

Option #2: <br>
1. Open **Extensions** sidebar panel in VS Code. `Press ctrl + shift + x`
2. Search for `Florist Theme`
3. Click **Install** to install it
